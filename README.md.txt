	1.1 正常存车是否能行。
	1.2 车位满时。能否正常停车
	1.3 相同的车牌号能否存上。
	2.取车。
	2.1 正常取车是否正常执行。
	2.2 车位空时。看能否取出车
	3.指示牌显示剩余空车位。
	3.1 车位为空时。显示是否正确。
	3.2 添加一个车辆时，数量能否正常显示。
	3.3 取一个车辆时，数量能否正常显示。
